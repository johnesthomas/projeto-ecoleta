## O que aprendi sobre em geral ?

### Primeiramente gostaria de agradecer pela oportunidade e dedicação de todos os colaboradoes e a comunidade que sempre esteve presente no discord tirando as dúvidas dos colegas. Assim, já se introduzindo em um ambiente de trabalho em equipe.

### O meu conhecimento de desenvolvimento é muito baixo, no segundo dia quase pensei em desistir, pois tudo era novo para mim. Mas as aulas seguintes foram ficando mais dinâmicas e as coisas foram se encaixando, e começei a tomar gosto pela coisa. 

### Usamos até uma API do IBGE para implementar o sistema de Estados e Cidades em forma de lista. 

### Outro conhecimento novo que tive foi em relação ao desenvolvimento Web Responsivo, com a adaptação do site para diferentes dispositivos, e com a implementação de textos nas imagens e por trás das mensagens, para que as pessoas com problemas auditivos possam acessar o site através do leitor dos browsers.

## O que aprendi sobre a parte técnica ?

### Os conhecimentos técnicos abordados no bootcamp foi totalmente focado na apredizagem, e tudo novo para mim. Mas consegui aprender bastante coisa. 

### Conhecemos o protocolo HTTP, como camuflar a URL, interação do Frontend com o Backend usando NodeJS e o banco de dados SQLite3, armazemento de dados no banco. O conhecimento foi grande, usando Linguagens de Programação atuais, linguagens de Banco de Dados e usando o próprio Javascript no backend do projeto.
